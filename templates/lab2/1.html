Get Started with Cloud Storage

APIs Explorer: Cloud Storage

Execute 1: Copy Paste the code in terminal.


gcloud auth list && gcloud config list project
export PROJECT_ID=$(gcloud config get-value project)

gsutil mb gs://$PROJECT_ID-bucket
gsutil mb gs://$PROJECT_ID-bucket0

curl -o demo-image1.png "https://cdn.qwiklabs.com/E4%2BSx10I0HBeOFPB15BFPzf9%2F%2FOK%2Btf7S0Mbn6aQ8fw%3D"
curl -o demo-image2.png "https://cdn.qwiklabs.com/Hr8ohUSBSeAiMUJe1J998ydGcTu%2FrF4BUjZ2J%2BbiKps%3D"

gsutil cp demo-image1.png gs://$PROJECT_ID-bucket/
gsutil cp demo-image2.png gs://$PROJECT_ID-bucket/

gsutil cp gs://$PROJECT_ID-bucket/demo-image1.png gs://$PROJECT_ID-bucket0/demo-image1-copy.png
