Get Started with Cloud Storage

Google Cloud Storage - Bucket Lock

Execute 1: Enter the value of Region displayed on the lab page, after clicking on start, For example: REGION=us-central1


REGION=    #write region from the code

Execute2: Copy Paste this code in the terminal


gcloud auth list && \
gcloud config list project && \
gcloud config set compute/region $REGION && \
export BUCKET=$(gcloud config get-value project) && \
gsutil mb "gs://$BUCKET" && \
gsutil retention set 10s "gs://$BUCKET" && \
gsutil retention get "gs://$BUCKET" && \
gsutil cp gs://spls/gsp297/dummy_transactions "gs://$BUCKET/" && \
gsutil ls -L "gs://$BUCKET/dummy_transactions" && \
yes | gsutil retention lock "gs://$BUCKET/"

Click on Check Box

Execute 3: Copy Paste this code in the terminal


sleep 15
gsutil retention temp set "gs://$BUCKET/dummy_transactions" && \
gsutil rm "gs://$BUCKET/dummy_transactions"
gsutil retention temp release "gs://$BUCKET/dummy_transactions"

Click on Check Box

Execute 4: Copy Paste this code in the terminal


gsutil rm "gs://$BUCKET/dummy_transactions" && \
gsutil retention event-default set "gs://$BUCKET/" && \
gsutil cp gs://spls/gsp297/dummy_loan "gs://$BUCKET/" && \
gsutil ls -L "gs://$BUCKET/dummy_loan" && \
gsutil retention event release "gs://$BUCKET/dummy_loan" && \
gsutil ls -L "gs://$BUCKET/dummy_loan"
