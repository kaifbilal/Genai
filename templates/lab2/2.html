Get Started with Cloud Storage

Cloud Storage: Qwik Start - CLI/SDK

Execute 1: Enter the value of Region displayed on the lab page, after clicking on start, For example: REGION=us-central1


REGION=    #write region from the code

Execute2: Copy Paste this code in the terminal


gcloud auth list && \
gcloud config list project && \
gcloud config set compute/region $REGION && \
export PROJECT_ID=$(gcloud config get-value project) && \
gsutil mb gs://$PROJECT_ID-bucket && \
curl https://upload.wikimedia.org/wikipedia/commons/thumb/a/a4/Ada_Lovelace_portrait.jpg/800px-Ada_Lovelace_portrait.jpg --output ada.jpg && \
gsutil cp ada.jpg gs://$PROJECT_ID-bucket && \
rm ada.jpg && \
gsutil cp -r gs://$PROJECT_ID-bucket/ada.jpg . && \
gsutil cp gs://$PROJECT_ID-bucket/ada.jpg gs://$PROJECT_ID-bucket/image-folder/ && \
gsutil ls gs://$PROJECT_ID-bucket && \
gsutil ls -l gs://$PROJECT_ID-bucket/ada.jpg && \
gsutil acl ch -u AllUsers:R gs://$PROJECT_ID-bucket/ada.jpg
